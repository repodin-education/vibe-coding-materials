## Overview

### Course Duration

**Total:** 6–10 hours of student work

**Options:**
- **One Week Intensive:** 6–10 hours over 7 days
- **Two Weeks Part-Time:** 6–10 hours over 14 days
- **Extended (3 Weeks):** 6–10 hours + pro-level extensions

## Module Breakdown

### Module 1: Intro to Cursor + GitHub

**Duration:** 1 hour

**Activities:**
- Read "Building with Cursor" guide
- Understand AI coding workflow
- Review Git basics

**Deliverable:** None (reading only)

**Timeline:**
```
┌─────────────────────────────────────┐
│ Module 1: Intro (1h)                │
├─────────────────────────────────────┤
│ Reading: 1h                         │
│ Deliverable: None                   │
└─────────────────────────────────────┘
```

### Module 2: Vibe Coding Fundamentals

**Duration:** 2–3 hours

**Activities:**
- Study prompting techniques
- Learn multi-file reasoning
- Understand iteration loops
- Complete reflection assignment

**Deliverable:** Assignment 1 (answers.md)

**Timeline:**
```
┌─────────────────────────────────────┐
│ Module 2: Fundamentals (2-3h)      │
├─────────────────────────────────────┤
│ Study: 1-2h                        │
│ Assignment 1: 1h                   │
│ Deliverable: answers.md            │
└─────────────────────────────────────┘
```

### Module 3: E2E Hello World

**Duration:** 2–4 hours

**Activities:**
- Set up development environment
- Create server (Node.js or Python)
- Create HTML client
- Test end-to-end
- Document process

**Deliverable:** Assignment 2 (working app)

**Timeline:**
```
┌─────────────────────────────────────┐
│ Module 3: E2E Hello World (2-4h)   │
├─────────────────────────────────────┤
│ Setup: 30min                        │
│ Development: 2-3h                   │
│ Testing: 1h                         │
│ Deliverable: Working app           │
└─────────────────────────────────────┘
```

### Module 4: Mini Feature

**Duration:** 1–2 hours

**Activities:**
- Plan feature
- Implement feature
- Test feature
- Document feature

**Deliverable:** Assignment 3 (extended app)

**Timeline:**
```
┌─────────────────────────────────────┐
│ Module 4: Mini Feature (1-2h)      │
├─────────────────────────────────────┤
│ Planning: 30min                     │
│ Development: 1-2h                   │
│ Deliverable: Extended app           │
└─────────────────────────────────────┘
```
