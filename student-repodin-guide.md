## Overview

### What is RepodIn Student Dashboard?

RepodIn Student Dashboard provides:
- **AI-powered feedback** on your code submissions
- **Progress tracking** across all assignments
- **Clear visibility** into your learning journey
- **Actionable insights** for improvement

### How It Works

1. You submit work in GitHub Classroom (normal workflow)
2. RepodIn automatically detects your submission
3. AI analysis runs automatically
4. You view feedback in RepodIn Student Dashboard
5. You improve based on feedback
6. You resubmit if needed

## Student Setup Steps

### Step 1: Create RepodIn Account

1. **Sign Up:**
   - Go to RepodIn website
   - Click **Sign up** or **Student Login**
   - Create account with your email
   - Verify email address

2. **Student Role:**
   - Select **Student** role during signup
   - Complete profile setup

### Step 2: Connect GitHub Account

1. **Connect GitHub:**
   - In RepodIn Student Dashboard, go to **Settings**
   - Navigate to **GitHub Integration**
   - Click **Connect GitHub Account**
   - Authorize RepodIn to access your GitHub account

2. **Verify Connection:**
   - Check GitHub username matches
   - Verify repository access
   - Connection confirmed

### Step 3: Access Course

1. **Automatic Enrollment:**
   - Course appears automatically when synced by teacher
   - No manual enrollment needed

2. **Manual Enrollment (if needed):**
   - Go to **Courses**
   - Search for "Vibe Coding"
   - Click **Enroll** if available

### Step 4: View Submissions

1. **Automatic Detection:**
   - Submissions from GitHub Classroom appear automatically
   - Matched by GitHub username or email

2. **View Submissions:**
   - Go to **Submissions** in dashboard
   - See all your submissions
   - Check status (pending, processing, completed)

### Step 5: Receive Feedback

1. **Automatic Analysis:**
   - AI analysis runs automatically on submission
   - Takes 2-5 minutes to complete

2. **View Feedback:**
   - Get notification when feedback is ready
   - View in Student Dashboard
   - Read detailed analysis

## Troubleshooting

### Common Issues

**Issue: Can't see my course**

**Solutions:**
- Verify teacher has synced course to RepodIn
- Check GitHub account is connected
- Verify GitHub username matches
- Contact teacher if still not visible

**Issue: Submission not appearing**

**Solutions:**
- Verify you pushed to GitHub
- Check GitHub username matches
- Wait a few minutes for sync
- Contact teacher if persists

**Issue: No feedback received**

**Solutions:**
- Check analysis status (may still be processing)
- Wait 2-5 minutes for analysis
- Verify submission was detected
- Contact teacher if no feedback after 10 minutes

**Issue: Can't connect GitHub account**

**Solutions:**
- Verify GitHub account is active
- Check permissions granted
- Try disconnecting and reconnecting
- Contact support if needed

### Getting Help

- Check RepodIn documentation
- Contact RepodIn support
- Ask teacher for help
- Use help channel
